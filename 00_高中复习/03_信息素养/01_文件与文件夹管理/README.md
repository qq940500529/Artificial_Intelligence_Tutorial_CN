# 文件与文件夹管理

## 概述

文件与文件夹是我们与计算机打交道时最基本的组织单元——无论是写一份文档、保存一张图片，还是运行一段代码，最终都要落实到磁盘上的某个文件中。学会高效地管理文件与文件夹，是一切后续学习的地基。如果连"文件存在哪里""叫什么名字""怎么找到它"这些问题都不清楚，后面学习编程、处理数据时就会频繁遇到"文件找不到""路径写错""数据被覆盖"等令人头疼的问题。

在人工智能的学习与实践中，文件管理能力的重要性尤为突出：训练数据集往往包含成千上万个文件，模型权重需要按版本妥善保存，实验结果需要有条理地归档以便日后对比。本主题从最基础的路径与扩展名讲起，逐步介绍命名规范、版本备份和压缩归档四个核心技能，帮助你在正式进入编程和数据处理之前，养成良好的文件管理习惯。这些习惯将直接迁移到后续的版本控制（Git）、数据管道搭建和实验管理等高级课程中。

## 学习建议

**学习顺序**：建议按照 01 → 02 → 03 → 04 的顺序依次学习。路径与扩展名是理解文件系统的基础，命名规范建立在对路径的理解之上，版本备份需要你已经掌握文件的命名和组织方式，最后的压缩与归档则是对前三个知识点的综合运用。

**时间分配**：本主题内容贴近日常操作，整体难度较低。建议每个知识点分配 20–30 分钟，总计约 1.5–2 小时即可完成。重点在于动手实践——边学边在自己的电脑上创建目录、重命名文件、尝试压缩和解压，比单纯阅读效果好得多。

**常见困难**：初学者最容易忽略的是绝对路径与相对路径的区别，以及不同操作系统（Windows 与 macOS/Linux）在路径分隔符上的差异。建议在学习路径知识点时多做练习，确保真正理解后再继续。
