# 表格与数据处理

> **所属路径**：`00_高中复习/03_信息素养/03_表格与数据处理`
> **包含知识点**：4 个

---

## 概述

表格是将零散信息转化为结构化数据的最常见方式——无论是记录一次考试的成绩、统计一个月的开支，还是整理一份调查问卷的结果，我们几乎都会自然而然地使用表格来组织数据。掌握表格与数据处理技能，就是在原始信息与有条理的分析之间架起一座桥梁，让数据从"一堆数字"变成"有意义的洞察"。

在人工智能领域，数据是驱动一切模型运转的燃料。学会在表格中规范地录入数据、灵活地排序与筛选、使用公式进行自动计算、通过图表直观地呈现规律，这些看似基础的操作，实际上为后续学习 NumPy、Pandas 等数值计算工具以及 SQL 查询语言奠定了直觉基础。更进一步，特征工程和探索性数据分析（EDA）中的核心思路——清洗、变换、聚合、可视化——都能在表格操作中找到最初的影子。本主题将带你从数据录入开始，一步步建立起完整的表格数据处理能力。

## 学习建议

建议按照 01 → 02 → 03 → 04 的顺序依次学习，整体预计需要 2 至 2.5 小时。四个知识点的难度循序渐进：先学会如何规范地"把数据放进表格"，再掌握排序与筛选来"快速找到数据"，然后用公式"让表格自动算数"，最后通过透视表和图表"让数据说话"。

学习过程中常见的困难包括：混淆相对引用与绝对引用（例如拖动公式时结果不符合预期）、不清楚何时该用柱状图而何时该用折线图等。遇到这些问题时不必着急，在对应的知识点课程中都会有详细的讲解和示例。建议每学完一个知识点后，立即用一份真实数据（如班级成绩表或家庭账单）动手练习，这样理解会更加深刻。
