# 概率基础

> **所属路径**：`00_高中复习/01_数学基础/09_概率基础`
> **包含知识点**：5 个

---

## 概述

概率是用数学语言描述"不确定性"的工具。在人工智能领域，几乎所有模型都在与不确定性打交道——从朴素贝叶斯分类器到深度学习中的随机梯度下降，概率思维贯穿始终。本主题从最直观的古典概率出发，逐步引入条件概率、独立事件、全概率公式与贝叶斯公式，最后初步认识随机变量，为后续系统学习概率论与统计打下坚实基础。

## 学习建议

1. **学习顺序**：严格按 01 → 02 → 03 → 04 → 05 顺序学习。古典概率建立基本概念，条件概率和独立事件是贝叶斯公式的前置，随机变量是统计学的入口。
2. **时间分配**：每个知识点约 30–50 分钟，整个主题约 3–4 小时。
3. **常见困难**：条件概率与独立事件的区别容易混淆，建议多画树形图辅助理解。
