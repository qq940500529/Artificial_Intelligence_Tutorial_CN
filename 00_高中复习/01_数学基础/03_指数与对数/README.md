# 指数与对数

> **所属路径**：`00_高中复习/01_数学基础/03_指数与对数`

---

## 概述

指数与对数是高中数学中最具应用价值的函数家族之一。指数描述了"成倍增长或衰减"的规律，对数则是指数的"反向操作"——用来回答"要乘多少次才能达到某个值"的问题。在人工智能领域，这两个概念的重要性怎么强调都不为过：softmax 函数依赖指数运算将模型输出转化为概率分布，交叉熵损失函数的核心是对数运算，信息熵用对数度量不确定性，学习率衰减遵循指数规律——可以说，指数与对数是贯穿整个 AI 数学体系的基本工具。

本主题从指数运算律出发，建立指数运算的规则体系；接着引入对数运算律，掌握对数的化简技巧；然后通过指数对数互化，打通两者之间的桥梁；最后分别研究指数函数和对数函数的图像与性质，为后续学习微积分中的指数增长模型和对数变换打下坚实基础。

## 学习建议

1. **学习顺序**：建议按 01 → 02 → 03 → 04 → 05 的顺序学习。指数运算律为对数运算律提供基础，指数对数互化将两者打通，最后指数函数和对数函数从图像角度深化理解。
2. **时间分配**：每个知识点预计 40–50 分钟，整个主题约 3.5–4 小时。
3. **常见困难**：很多学习者容易混淆对数运算律中的"积的对数"与"对数的积"，建议在对数运算律这一节多做练习。指数函数和对数函数互为反函数的关系，可以通过画图辅助理解。
