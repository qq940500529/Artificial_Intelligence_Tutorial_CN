## 概述

数列是数学中研究"按规律排列的数"的学科。它既是高中代数的核心内容，也是理解人工智能的重要基础。等差数列对应线性增长模式，等比数列对应指数增长与衰减模式——这两种模式在 AI 中无处不在：学习率调度、梯度传播、折扣累积奖励等都与数列紧密相关。递推关系则是循环神经网络和自回归模型的数学原型，而数列极限的概念直接对应训练过程中"收敛"的含义。

本主题从等差数列出发，建立"等步长增长"的直觉；接着学习等比数列，理解"等比例增长"的威力与危险（指数爆炸或指数衰减）；然后通过递推与求和，掌握更复杂数列的分析工具；最后引入数列极限的概念，为后续学习微积分打下基础。

## 学习建议

1. **学习顺序**：建议按 01 → 02 → 03 → 04 的顺序学习。等差数列最基础，等比数列在等差数列基础上引入乘法递推，递推与求和是对前两者的泛化，数列极限为微积分做准备。
2. **时间分配**：每个知识点预计 50 分钟，整个主题约 3.5 小时。
3. **常见困难**：求和公式容易记混（等差用"首尾平均"，等比用"错位相减"），建议理解推导过程而非死记公式。极限的概念初学时可能觉得抽象，建议结合 Python 代码的数值验证来建立直觉。
