> **所属路径**：`00_高中复习/02_英语基础/01_技术词汇`
> **包含知识点**：4 个

---

## 概述

在人工智能的学习旅程中，你会不断遇到英文术语——从数学公式中的 variable（变量）、function（函数），到编程中的 loop（循环）、debug（调试），再到 AI 领域的 neural network（神经网络）、machine learning（机器学习）。如果每次遇到新词都要停下来查词典，学习效率会大打折扣。

本主题的目标是帮助你在正式进入人工智能学习之前，建立一套**核心技术词汇表**。我们将词汇按领域分为四组：数学、编程、人工智能和机器学习。每组词汇都配有中文释义、发音提示和例句，帮助你在真实技术场景中快速识别和理解这些术语。掌握这些词汇后，你将能更顺畅地阅读英文文档、理解报错信息，并为后续的技术英语学习打下坚实基础。

## 学习建议

**推荐学习顺序**：按照 01 → 02 → 03 → 04 的顺序学习。数学词汇和编程词汇是基础，AI 词汇和机器学习术语建立在前两者之上。

**时间分配**：每个知识点建议花 30–45 分钟完成首次学习，之后利用碎片时间反复回顾。词汇学习的关键是**多次重复**，建议在后续课程学习中不断回查本主题的词汇表。

**常见困难**：
- **发音障碍**：不必追求标准发音，能在脑中建立"看到拼写→联想含义"的映射即可。
- **记忆遗忘**：属于正常现象，建议制作单词卡片（参见 [总结与记笔记](../04_总结与记笔记/) 中的双语术语卡片方法）。
- **词汇混淆**：部分近义词容易混淆（如 parameter 和 argument），课程中会专门区分。
