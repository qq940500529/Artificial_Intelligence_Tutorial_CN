## 概述

Jupyter Notebook 是数据科学和人工智能领域最重要的交互式开发工具之一。它将代码编写、运行结果、可视化图表和文字说明融合在同一个文档中，形成了一种"可执行的实验记录"。无论是探索一份新数据集、调试一个机器学习模型，还是撰写一份技术报告，Notebook 都能让你在"写一点、跑一点、看一点"的循环中快速迭代，极大地降低了从想法到验证的门槛。

对于初学者来说，Jupyter Notebook 是进入 Python 数据分析与 AI 世界的最佳起点——你不需要一次性写完整个程序，而是可以逐步探索、即时查看结果、随时修改方向。本主题将从 Notebook 的基本操作出发，逐步覆盖可视化展示、实验记录与导出、魔法命令与扩展等核心能力，帮助你建立高效的交互式开发工作流。

## 学习建议

建议按照知识点编号顺序学习，总计约 2.5 小时：

1. **先打好基础**（01，约 35 分钟）：从单元格与执行顺序开始，理解 Notebook 的核心操作模型。这是后续所有操作的基础，务必动手操作每一个示例。
2. **学会展示**（02，约 45 分钟）：掌握可视化与富文本展示能力，这是 Notebook 区别于普通脚本的核心优势。
3. **形成工作流**（03，约 40 分钟）：学习如何将 Notebook 组织为规范的实验记录，以及如何导出和分享。
4. **提升效率**（04，约 35 分钟）：通过魔法命令和扩展进一步提升开发效率。

**常见困难点**：
- 非线性执行导致的隐藏状态问题（01 中重点讲解）
- Notebook 与版本控制的冲突（03 中重点讲解）
- 魔法命令与普通 Python 代码的区别（04 中重点讲解）

**建议**：学习本主题时，请务必打开一个 Jupyter Notebook 实际操作。仅靠阅读无法真正掌握交互式开发的工作流。
